# Crud-Node-React-Mongo
NodeJS + ReactJS + MongoDB

### Dependencies
```shell
npm i express
npm i cookie-parser
npm i cors
npm i path
npm i nodemon -d
npm i mongoose
npm i bcrypt

npm run dev
```

### Swagger
```shell
npm install --save-dev swagger-autogen
npm install swagger-ui-express
npm run swagger-autogen
```
